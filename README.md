## 1.1 C언어란 무엇인가?

- **C언어**: 인간과 컴퓨터 사이의 의사소통을 위한 프로그래밍 언어
- **프로그래밍 언어의 종류**: C, C++, C#, Java 등
- **프로그램**: 프로그래밍 언어로 작성한 작업 결과물
- **프로그래머**: 프로그래밍을 하는 사람

### C언어의 탄생

- **제작자**: 미국 AT&T 벨(Bell) 연구소의 켄 톰슨, 데니스 리치
- **제작 목적**: 프로그램의 이식성을 높이기 위해

```
ALGOL 60 → CPL → BCPL → B → C
```

---

## 1.2 컴파일러란 무엇인가?

- **컴파일러**: 인간이 만든 프로그램을 기계가 이해할 수 있도록 **기계어로 변환**하는 변환기 (통역관 역할)
- **기계어**: 기계가 이해하는 2진 숫자(0과 1)로 작성된 언어

```
인간 (C 언어) → [C 컴파일러] → 기계 (기계어)
```

---

## 1.3 프로그램 작성 방법 4단계

### 이론

| 단계 | 수행 주체 | 입력 | 출력 |
|------|-----------|------|------|
| Step 1 | 인간 | - | 소스 파일 (`.c`) |
| Step 2 | 컴파일러 | `.c` | 오브젝트 파일 (`.obj`) |
| Step 3 | 링커 | `.obj` + 라이브러리 | 실행 파일 (`.exe`) |
| Step 4 | 로더 | `.exe` | 실행 (주기억 장치에 로딩) |

```
프로그램 작성 → *.c → [컴파일] → *.obj → [링크] → *.exe → [실행]
```

> Step 1 ~ Step 4까지의 작업을 도와주는 소프트웨어를 **통합 개발 소프트웨어(IDE)** 라고 한다.

### 실습

**Step 1 — 소스 파일 작성 (Hello.c)**

```c
#include <stdio.h>

int main(void)
{
    printf("Hello C world \n");
    return 0;
}
```

**Step 2 — 컴파일 (Hello.obj 생성)**

- Visual Studio: `빌드(B)` 메뉴 → `컴파일(M)` (`Ctrl+F7`)
- 결과: `HelloWorld.obj` 파일 생성

**Step 3 — 링크 (HelloWorld.exe 생성)**

- Visual Studio: `빌드(B)` 메뉴 → `솔루션 빌드(B)` (`Ctrl+Shift+B`)
- 결과: `HelloWorld.exe` 파일 생성

**Step 4 — 실행**

- Visual Studio: `디버그(D)` 메뉴 → `디버그하지 않고 시작(H)` (`Ctrl+F5`)
- 결과: 콘솔 출력 `Hello, World!`
