## 1.1 C언어란 무엇인가?

- **C언어**: 인간과 컴퓨터 사이의 의사소통을 위한 프로그래밍 언어
- **프로그래밍 언어의 종류**: C, C++, C#, Java 등
- **프로그램**: 프로그래밍 언어로 작성한 작업 결과물
- **프로그래머**: 프로그래밍을 하는 사람

### C언어의 탄생

- **제작자**: 미국 AT&T 벨(Bell) 연구소의 켄 톰슨, 데니스 리치
- **제작 목적**: 프로그램의 이식성을 높이기 위해

```
ALGOL 60 → CPL → BCPL → B → C
```

---

## 1.2 컴파일러란 무엇인가?

- **컴파일러**: 인간이 만든 프로그램을 기계가 이해할 수 있도록 **기계어로 변환**하는 변환기 (통역관 역할)
- **기계어**: 기계가 이해하는 2진 숫자(0과 1)로 작성된 언어

```
인간 (C 언어) → [C 컴파일러] → 기계 (기계어)
```

---

## 1.3 프로그램 작성 방법 4단계

### 이론

| 단계 | 수행 주체 | 입력 | 출력 |
|------|-----------|------|------|
| Step 1 | 인간 | - | 소스 파일 (`.c`) |
| Step 2 | 컴파일러 | `.c` | 오브젝트 파일 (`.obj`) |
| Step 3 | 링커 | `.obj` + 라이브러리 | 실행 파일 (`.exe`) |
| Step 4 | 로더 | `.exe` | 실행 (주기억 장치에 로딩) |

```
프로그램 작성 → *.c → [컴파일] → *.obj → [링크] → *.exe → [실행]
```

> Step 1 ~ Step 4까지의 작업을 도와주는 소프트웨어를 **통합 개발 소프트웨어(IDE)** 라고 한다.

### 실습

**Step 1 — 소스 파일 작성 (Hello.c)**

```c
#include <stdio.h>

int main(void)
{
    printf("Hello C world \n");
    return 0;
}
```

**Step 2 — 컴파일 (Hello.obj 생성)**

- Visual Studio: `빌드(B)` 메뉴 → `컴파일(M)` (`Ctrl+F7`)
- 결과: `HelloWorld.obj` 파일 생성

**Step 3 — 링크 (HelloWorld.exe 생성)**

- Visual Studio: `빌드(B)` 메뉴 → `솔루션 빌드(B)` (`Ctrl+Shift+B`)
- 결과: `HelloWorld.exe` 파일 생성

**Step 4 — 실행**

- Visual Studio: `디버그(D)` 메뉴 → `디버그하지 않고 시작(H)` (`Ctrl+F5`)
- 결과: 콘솔 출력 `Hello, World!`

# 제2장 C 언어의 기본 구조와 표준 입출력

## 2.1 C언어의 기본 구조

C 프로그램은 크게 3가지 구역으로 나뉜다.

```c
/*
    파일명: Hello.c
    만든이: 홍길동
    프로그램 내용: Hello C world 테스트
*/                          // ← Step 1: 주석

#include <stdio.h>          // ← Step 2: 전처리기 + 헤더파일

int main(void)              // ← Step 3: main() 함수 시작
{
    printf("Hello C world \n");
    return 0;
}                           // ← Step 3: main() 함수 끝
```

---

### Step 1 — 주석 (Comment)

**주석이란?**
- 코드 안에 메모를 남기는 기능
- 컴파일러는 주석의 내용을 완전히 무시함 → 실행에 영향 없음
- 코드를 설명하거나, 임시로 코드를 비활성화할 때 사용

**주석 작성 방법 2가지**

```c
/* 여러 줄 주석
   이렇게 여러 줄을 한 번에 주석 처리할 수 있다 */

// 한 줄 주석: 이 줄 한 줄만 주석 처리된다
```

**⚠️ 주의: `/* */` 중복 사용 금지**

```c
// ❌ 오류 발생: /* 안에 또 /* 를 쓰면 안 됨
/*
    /* 파일명: Hello.c */
    만든이: 홍길동
*/

// ✅ 올바른 방법: /* */ 안에서는 // 를 사용
/*
    // 파일명: Hello.c
    // 만든이: 홍길동
*/
```

> `/* */`는 첫 번째 `*/`를 만나는 순간 주석이 끝난다고 판단하기 때문에 중복 사용 시 오류가 발생한다.

---

### Step 2 — 전처리기와 헤더파일

```c
#include <stdio.h>
①       ②③
```

각 구성요소의 의미:

| 기호 | 이름 | 의미 |
|------|------|------|
| `#` | 전처리기 (Preprocessor) | 컴파일 전에 먼저 처리하라는 신호 |
| `include` | 포함하다 | 특정 파일을 소스코드에 포함시키는 명령 |
| `<stdio.h>` | 헤더파일 (Header File) | 확장자 `.h`를 가진 파일. `stdio`는 **Standard Input Output**의 약자 |

**한 문장으로 해석하면:**
> "컴파일러야, 컴파일 전에 `stdio.h` 파일을 먼저 포함시켜 다오!"

**`stdio.h`가 왜 필요한가?**
- `printf()`, `scanf()` 같은 함수들은 C언어에 기본으로 내장된 것이 아님
- 이 함수들의 정의가 `stdio.h` 파일 안에 들어 있음
- `#include <stdio.h>` 를 써야만 `printf()`, `scanf()` 를 사용할 수 있음

---

### Step 3 — main() 함수

```c
int main(void)   // 출력형태: int / 함수이름: main / 입력형태: void
{                // 함수의 시작
    printf("Hello C world \n");
    return 0;
}                // 함수의 끝
```

**각 키워드 의미:**

| 키워드 | 의미 |
|--------|------|
| `int` | integer(정수). main 함수가 끝날 때 정수를 반환한다는 뜻 |
| `main` | 함수의 이름. C 프로그램은 무조건 main에서 시작됨 |
| `void` | 비어있다(빈 공간). 입력받는 값이 없다는 뜻 |
| `{ }` | 함수의 범위(시작과 끝)를 표시 |

**main() 함수는 누가 호출하는가?**
- **운영체제(OS)** 가 프로그램 실행 시 맨 처음 호출하고, 맨 나중에 종료시킨다.

**`return 0;` 의미:**

```c
return 0;   // 운영체제에게 0을 돌려주고, main() 함수를 종료해라
return;     // 반환할 값 없이 그냥 main() 함수를 종료해라
```

> `0`을 반환하는 것은 "프로그램이 정상적으로 종료되었다"는 신호를 운영체제에 보내는 것이다.

**세미콜론 `;` 의 역할:**
- 문장의 끝을 알리는 마침표와 같은 역할
- C언어에서 모든 실행문 끝에는 반드시 `;` 을 붙여야 한다

```c
printf("Hello C world \n");   // ← 세미콜론 필수!
return 0;                     // ← 세미콜론 필수!
```

## 2.3 모니터에 데이터 출력하기 — `printf()` 함수

### 특수 문자 (Escape Sequence)

특수 문자란 `\`(역슬래시)로 시작하는 특별한 의미를 가진 문자다.

```c
printf("Hello C world \n");   // \n 이 있으면 출력 후 줄바꿈
printf("Hello C world");      // \n 없으면 줄바꿈 없이 바로 이어짐
```

| 특수 문자 | 의미 |
|-----------|------|
| `\n` | 개행 (New Line) — 줄바꿈 |
| `\t` | 수평 탭 (Tab키 효과) |
| `\\` | 역슬래시 `\` 출력 |
| `\"` | 큰따옴표 `"` 출력 |
| `\'` | 작은따옴표 `'` 출력 |
| `\a` | 경고음 소리 |
| `\b` | 백스페이스 |
| `\r` | 캐리지 리턴 |

**특수 문자 예제 (2-3.c):**

```c
#include <stdio.h>

int main(void)
{
    printf("\t Hello C! \n Hello World! \n");
    // \t → 탭, \n → 줄바꿈

    printf("큰따옴표: \" \" \n");   // \" → " 출력
    printf("작은따옴표: \' \' \n"); // \' → ' 출력
    printf("역슬래시: \\ \n");      // \\ → \ 출력

    return 0;
}
```

**출력 결과:**
```
    Hello C!
Hello World!
큰따옴표: " "
작은따옴표: ' '
역슬래시: \
```

### 서식 문자 (Format Specifier)

`printf()`의 `f`는 **Formatted(서식화된)** 의 약자다.
서식 문자는 어떤 형태로 값을 출력할지 지정하는 `%`로 시작하는 기호다.

```c
printf("%d 더하기 %d 는 %d 입니다\n", 3, 5, 3+5);
//      ↑           ↑       ↑          ↑  ↑  ↑
//      서식문자들이 순서대로 뒤의 값들과 매칭됨
// 출력: 3 더하기 5 는 8 입니다
```

**출력 서식 문자 전체 목록:**

| 서식 문자 | 출력 형태 |
|-----------|-----------|
| `%d`, `%i` | 10진수 정수 (양수/음수 모두 가능) |
| `%u` | 10진수 정수 (양수만) |
| `%x` | 16진수 정수 (양수만) |
| `%o` | 8진수 정수 (양수만) |
| `%f` | 10진수 실수 (소수점 이하 6자리) |
| `%lf` | 10진수 실수 double형 |
| `%e` | e 표기법 실수 (소문자 e) |
| `%E` | E 표기법 실수 (대문자 E) |
| `%g` | 자릿수에 따라 `%f` 또는 `%e` 자동 선택 |
| `%G` | 자릿수에 따라 `%f` 또는 `%E` 자동 선택 |
| `%c` | 문자 1개 |
| `%s` | 문자열 |
| `%%` | `%` 기호 자체를 출력 |

---

#### ① 정수 출력 예제 (2-4.c, 2-5.c)

```c
#include <stdio.h>

int main()
{
    // %d와 %i는 동일하게 10진수 정수 출력
    printf("%d 더하기 %d 는 %d 입니다\n", 3, 5, 3+5);
    printf("%i 더하기 %i 는 %i 입니다\n", 3, 5, 3+5);

    // 음수도 표현 가능
    printf("%d - %d = %d 입니다\n", 3, 5, 3-5);

    return 0;
}
// 출력:
// 3 더하기 5 는 8 입니다
// 3 더하기 5 는 8 입니다
// 3 - 5 = -2 입니다
```

```c
// 16진수, 8진수 출력 (음수 표현 불가)
printf("10진수: %d는 16진수: %x, 8진수: %o 입니다.\n", 50, 50, 50);
// 출력: 10진수: 50는 16진수: 32, 8진수: 62 입니다.

printf("10진수: %d는 16진수: %x, 8진수: %o 입니다.\n", -50, -50, -50);
// ⚠️ %x, %o는 음수 표현 불가 → 이상한 값 출력됨
```

#### ② 실수 출력 예제 (2-6.c)

```c
#include <stdio.h>

int main(void)
{
    printf("10진수 정수: %d \n", 0.5);   // ⚠️ 실수를 %d로 출력 → 0 출력 (잘못된 사용)
    printf("10진수 실수: %f \n", 0.5);   // 0.500000 출력 (소수점 이하 6자리 기본)
    printf("10진수 실수: %lf \n", 0.5);  // 0.500000 출력

    printf("소수점 이하 6자리 이상: %f \n", 0.5655678);   // 0.565568 (7번째 자리에서 반올림)
    printf("소수점 이하 6자리 이상: %lf \n", 0.5667784);  // 0.566778

    return 0;
}
```

> `%f`는 기본적으로 소수점 이하 **6자리**까지만 출력한다. 7번째 자리에서 반올림.

#### ③ 문자/문자열 출력 예제 (2-7.c)

```c
#include <stdio.h>

int main(void)
{
    printf("a를 대문자로 표현하면 %c 입니다.\n", 'A');
    // %c → 문자 1개, 작은따옴표로 감쌈

    printf("%s %c 입니다.\n", "a를 대문자로 표현하면", 'A');
    // %s → 문자열, 큰따옴표로 감쌈

    printf("%s %d %s \n", "2곱하기3은", 2*3, "입니다.");
    // 출력: 2곱하기3은 6 입니다.

    return 0;
}
```

#### ④ %u 예제 (2-8.c)

```c
#include <stdio.h>

int main(void)
{
    printf("%d \n", 2147483647);   // int의 최댓값 → 2147483647
    printf("%d \n", 2147483650);   // 최댓값 초과 → 오버플로우 발생, -2147483646 출력
    printf("%u \n", 4294967295);   // %u는 양수 전용 → 더 큰 수 표현 가능

    return 0;
}
```

> `%d`는 음수/양수 모두 표현하므로 표현 범위가 약 ±21억까지다. `%u`는 양수 전용이라 약 42억까지 표현 가능하다.

#### ⑤ 기타 실수 표기법 예제 (2-9.c)

```c
#include <stdio.h>

int main(void)
{
    printf("%f \n", 0.000123);       // 0.000123   (소수점 이하 6자리)
    printf("%f \n", 0.0001236);      // 0.000124   (반올림)

    printf("%e \n", 0.000123);       // 1.230000e-004  (소문자 e 표기)
    printf("%E \n", 0.0001236);      // 1.236000E-004  (대문자 E 표기)

    printf("%g \n", 0.000123);       // 0.000123   (6자리 이하면 %f 방식)
    printf("%G \n", 0.0000123456);   // 1.23456E-005  (6자리 초과면 %E 방식)

    printf("올해 우리나라 경제 성장률은 5%% 다.\n");
    // %% → % 기호 하나를 출력

    return 0;
}
```

### 필드 폭 지정

출력할 값의 자릿수와 정렬 방식을 제어할 수 있다.

| 서식 | 의미 |
|------|------|
| `%3d` | 3칸 확보, 오른쪽 정렬 |
| `%-3d` | 3칸 확보, 왼쪽 정렬 |
| `%03d` | 3칸 확보, 오른쪽 정렬, 빈 자리는 `0`으로 채움 |
| `%+3d` | 3칸 확보, 오른쪽 정렬, 양수는 `+`, 음수는 `-` 표시 |

```c
#include <stdio.h>

int main(void)
{
    printf("%03d, %03d, %03d \n", 1, 20, 300);
    // 출력: 001, 020, 300

    printf("%-3d, %-3d, %-3d \n", 1, 20, 300);
    // 출력: 1  , 20 , 300  (왼쪽 정렬, 빈 자리는 공백)

    printf("%+3d, %+3d, %+3d \n", 1, 20, 300);
    // 출력:  +1, +20, +300

    printf("%+3d, %+3d, %+3d \n", 1, 20, -300);
    // 출력:  +1, +20, -300

    return 0;
}
```

## 2.4 키보드로 데이터 입력받기 — `scanf_s()` 함수

### printf() vs scanf()

| 구분 | `printf()` | `scanf_s()` |
|------|-----------|------------|
| 역할 | 모니터에 데이터 **출력** | 키보드로 데이터 **입력** |
| 헤더파일 | `stdio.h` | `stdio.h` |
| 이름 유래 | **Print** + **f**ormatted | **Scan** + **f**ormatted |

---

### scanf_s() 기본 구조

```c
scanf_s("%d", &a);
//       ①     ②
```

| 구성요소 | 설명 |
|----------|------|
| `%d` | 입력 서식 문자 (어떤 형식으로 입력받을지) |
| `&a` | 입력받은 값을 저장할 변수 (`&`는 변수의 주소를 의미) |

**해석:**
1. 키보드로부터 데이터를 `%d` (10진수 정수) 형식으로 입력받는다.
2. 입력받은 값을 변수 `a`에 저장한다.

> `&` (앰퍼샌드)는 "변수의 주소"를 의미한다. scanf는 값을 변수에 직접 저장해야 하므로 변수의 메모리 주소가 필요하다. 자세한 내용은 포인터 챕터에서 다룬다.

---

### 입력 서식 문자 전체 목록

| 서식 문자 | 입력 형태 |
|-----------|-----------|
| `%d` | 10진수 정수 |
| `%x` | 16진수 정수 |
| `%o` | 8진수 정수 |
| `%f` | float형 실수 |
| `%lf` | double형 실수 |
| `%e` | float형 e 표기법 실수 |
| `%le` | double형 e 표기법 실수 |
| `%c` | 문자 1개 (추후 학습) |
| `%s` | 문자열 (추후 학습) |
| `%u` | 10진수 양수 정수 (추후 학습) |

### scanf_s() 예제 (2-11.c)

```c
#include <stdio.h>

int main(void)
{
    int age;   // 정수형 변수 age 선언 (값을 담는 임시 공간)

    printf("What is your age?: ");  // 입력 안내 메시지 출력
    scanf_s("%d", &age);            // 키보드에서 정수 입력 → age에 저장
    printf("Wow! Really? Are you %d years old?\n", age);  // 저장된 값 출력

    return 0;
}
// 실행 예시:
// What is your age?: 20
// Wow! Really? Are you 20 years old?
```

### 정수 입력 예제 (2-12.c)

```c
#include <stdio.h>

int main(void)
{
    int a, b, c;

    printf("10진수 정수 1개 입력: ");
    scanf_s("%d", &a);
    printf("10진수: %d, 16진수: %x, 8진수: %o \n", a, a, a);
    // 입력: 10 → 출력: 10진수: 10, 16진수: a, 8진수: 12

    printf("16진수 정수 1개 입력: ");
    scanf_s("%x", &b);
    printf("10진수: %d, 16진수: %x, 8진수: %o \n", b, b, b);
    // 입력: 20 → 출력: 10진수: 32, 16진수: 20, 8진수: 40

    printf("8진수 정수 1개 입력: ");
    scanf_s("%o", &c);
    printf("10진수: %d, 16진수: %x, 8진수: %o \n", c, c, c);
    // 입력: 30 → 출력: 10진수: 24, 16진수: 18, 8진수: 30

    return 0;
}
```

### 실수 입력 예제 (2-13.c)

```c
#include <stdio.h>

int main(void)
{
    float f1, f2;      // float: 단정밀도 실수형
    double d1, d2;     // double: 배정밀도 실수형 (더 정밀)

    printf("float형 실수 두 개 입력: ");
    scanf_s("%f %e", &f1, &f2);           // 스페이스바로 구분해서 두 값 입력
    printf("float형 실수 출력: f1=%f, f2=%e \n", f1, f2);

    printf("double형 실수 두 개 입력: ");
    scanf_s("%lf %le", &d1, &d2);
    printf("double형 실수 출력: d1=%lf, d2=%le \n", d1, d2);

    return 0;
}
// 실행 예시:
// float형 실수 두 개 입력: 1.1 2.2
// float형 실수 출력: f1=1.100000, f2=2.200000e+000
```

> `float`와 `double`의 차이: `double`이 더 많은 소수점 자릿수를 저장할 수 있다. `float`은 `%f`·`%e`, `double`은 `%lf`·`%le` 서식 문자를 사용한다.

## 2.5 학습 내용 정리

- C 프로그램의 기본 구조 3단계: **주석 → 전처리기/헤더파일 → main() 함수**
- `#include <stdio.h>`: `printf()`, `scanf()`를 사용하기 위해 반드시 필요
- `printf()`: 모니터 출력 함수, 서식 문자 `%d %f %c %s` 등 사용
- `scanf_s()`: 키보드 입력 함수, 변수 앞에 `&`를 붙여야 함
- 특수 문자 `\n`, `\t`, `\\`, `\"` 등으로 출력 형태 제어 가능
- 서식 문자로 정수/실수/문자/문자열 등 다양한 형태로 출력/입력 가능
